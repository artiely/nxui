## API

| 属性   | 说明                                                                | 类型                  | 默认值 |
| ------ | ------------------------------------------------------------------- | --------------------- | ------ |
| name   | 图标名称                                                            | String `icon-clock`   | -      |
| size   | 图标大小                                                            | String `12px` `24rem` | 44px   |
| color  | 图标颜色(symbol 下无效)                                             | String                |        |
| symbol | svg 多色图标模式 默认 iconfont 单色图标模式 此模式下 color 属性无效 | Boolean               | false  |
